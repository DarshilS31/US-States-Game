A Python-based interactive game where users guess U.S. state names.
Correct guesses are displayed on the map using Turtle graphics.

Tech Used:
- Python
- Turtle
- Pandas

Features:
- Case-insensitive input handling
- Prevents duplicate guesses
- Shows state names at correct coordinates
- Ends game on cancel or after 50 correct guesses

How to Run:
1. Install dependencies (pandas)
2. Run main.py

Learning Outcome:
- Pandas DataFrame filtering
- Handling user input in loops
- Working with Turtle graphics
